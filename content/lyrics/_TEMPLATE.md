HOW TO ADD A SONG
-----------------
1. Make a copy of this file in the lyrics/ folder.
2. Rename it to your song, e.g. 3-the-song-name.md
   (the number just controls the order they appear in).
3. Change the title / year / listen link above the second "---".
4. Replace this text with the words.

(Files starting with "_" — like this one — are ignored by the build,
 so this template never shows up on the site.)
